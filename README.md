# Nikel - Controle Financeiro

Aplicação web para controle de finanças pessoais, desenvolvida como entrega prática do módulo de Git e PaaS.

O objetivo principal do projeto foi aplicar conceitos de versionamento de código, organização de diretórios e publicação (deploy) em ambiente de hospedagem.

## Sobre o Projeto

O Nikel permite que o usuário crie uma conta e gerencie suas finanças de forma simples. O sistema armazena as transações e atualiza o saldo em tempo real.

### Funcionalidades

* **Cadastro e Login:** Criação de conta de usuário e autenticação.
* **Transações:** Registro de entradas e saídas de valores.
* **Painel:** Visualização do saldo atual e lista das últimas movimentações.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git (Versionamento)
